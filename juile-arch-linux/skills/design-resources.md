# Design Resources (verified, world-class)

A curated library of top-tier, real frontend/design references. When a design task needs
depth or current best practice, `web_search` or `web_fetch` these rather than guessing.
All links are canonical sources (not scraped/copied content).

## Foundations & principles
- Refactoring UI — https://www.refactoringui.com/ — practical visual-design rules for developers.
- Laws of UX — https://lawsofux.com/ — UX heuristics (Fitts, Hick, Jakob, etc.) with examples.
- Nielsen Norman Group — https://www.nngroup.com/articles/ — evidence-based UX research.
- Smashing Magazine — https://www.smashingmagazine.com/ — deep, current frontend/design articles.

## Design systems & guidelines
- Material Design 3 — https://m3.material.io/ — Google's system: color, type, motion, components.
- Apple Human Interface Guidelines — https://developer.apple.com/design/human-interface-guidelines/
- Web.dev (Google) — https://web.dev/ — performance, UX, Core Web Vitals, modern CSS.
- MDN Web Docs — https://developer.mozilla.org/ — the reference for HTML/CSS/JS/ARIA.

## CSS & layout craft
- Every Layout — https://every-layout.dev/ — robust, resilient layout primitives.
- Josh W. Comeau — https://www.joshwcomeau.com/ — CSS, animation, and interaction deep-dives.
- Inclusive Components — https://inclusive-components.design/ — accessible component patterns.
- CSS-Tricks — https://css-tricks.com/ — broad CSS technique reference.
- Tailwind CSS docs — https://tailwindcss.com/docs — utility-first patterns + good defaults.

## Accessibility
- WCAG 2.2 Quick Reference — https://www.w3.org/WAI/WCAG22/quickref/
- WAI-ARIA Authoring Practices — https://www.w3.org/WAI/ARIA/apg/ — accessible widget patterns.

## Color, type & motion tools
- Type Scale — https://typescale.com/ — modular type scales.
- Coolors — https://coolors.co/ — palette generation.
- Realtime Colors — https://www.realtimecolors.com/ — preview palettes on a live UI.
- cubic-bezier — https://cubic-bezier.com/ — craft easing curves.
- Easings reference — https://easings.net/ — named easing functions with curves.

## Assets (check licenses)
- Google Fonts — https://fonts.google.com/ — open-source typefaces.
- Lucide icons — https://lucide.dev/ — clean MIT-licensed icon set.
- Heroicons — https://heroicons.com/ — MIT icons by the Tailwind team.
- unDraw — https://undraw.co/ — open illustrations.

## Inspiration (study, don't copy)
- Mobbin — https://mobbin.com/ — real product UI patterns.
- Godly — https://godly.website/ — high-end web design.
- Awwwards — https://www.awwwards.com/ — award-winning sites.

> Usage note: treat these as primary sources to cite and learn from. Respect each site's
> license/terms — link and reference, don't wholesale-copy their content into outputs.
