# Build a design system
1. Define tokens first: color (1 accent + neutrals + semantic), an 8pt spacing scale, a type scale (1.25 ratio), radii, shadows, motion (150-300ms ease-out).
2. Build primitives: Button (variants/sizes/states), Input, Card, Badge, Modal — each with hover/focus/active/disabled.
3. Document with a live preview page. Keep it consistent; consistency reads as quality.
