# Create a CLI tool

Category: Engineering

Use this skill to create a CLI tool.

1. Clarify the goal and constraints; state assumptions.
2. Gather inputs (ask the user with ask_user, or use tools).
3. Produce the result with the right tool or widget (chart/table/cards/steps/email/file/blueprint/xlsx).
4. Verify it, then summarise what you did in one short line.
