# Glassmorphism / liquid-glass UI
Frosted panels: translucent gradient fill (top brighter), backdrop-filter blur(28-36px) saturate(180%), NO hard border —
use an inset top highlight (inset 0 1px 0 rgba(255,255,255,.6)) and a soft outer shadow. Keep text high-contrast white.
Layer cards over a vivid gradient/photo background so the blur has something to refract.
