# Design a landing page that converts
1. Above the fold: one sharp headline, a one-line subhead, a primary CTA, and a real product visual. No fluff.
2. Sections: social proof, 3 feature blocks (icon + title + body), how-it-works steps, pricing, FAQ, footer CTA.
3. Bold type scale, generous whitespace, one accent color, subtle scroll/hover motion. Fully responsive. Fast.
